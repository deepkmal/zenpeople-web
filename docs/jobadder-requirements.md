Before the CRM integration can be built, the following items are needed from the JobAdder API team.

## 1. OAuth2 Client Credentials

- **Client ID** and **Client Secret** for the OAuth2 application
- Register the redirect URI: `https://zenpeople.com.au/api/jobadder/callback`
- Required scopes: `read write offline_access`

## 2. Website Job Board Portal

- A job board portal must be set up in JobAdder for the Zenpeople website
- This provides the **Board ID** needed to post applications against the correct job listings

## 3. API Access

- Confirm API access is enabled on the JobAdder account
- Confirm which JobAdder API version to target (v2)
